# AI-Startup-Idea-Generator# 🚀 AI Startup Idea Generator

An AI-powered web application built using **Python** and **Flask** that generates creative startup ideas based on the industry entered by the user.

---

## ✨ Features

* 🚀 Generate AI Startup Ideas
* 🌙 Dark & Light Mode
* ❤️ Save Favorite Startups
* 📄 Download Startup Report as PDF
* 🕒 Recent Search History
* 📊 Startup Score & Success Rate
* 📱 Responsive User Interface
* 🎨 Modern UI Design
* ⚡ Animated Progress Bar

---

## 🛠️ Tech Stack

* Python
* Flask
* HTML5
* CSS3
* JavaScript
* ReportLab

## ▶️ How to Run

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Run the project

python app.py

4. Open your browser

http://127.0.0.1:5000
